# vue-wc-async-poc

## Project setup

```
npm install
```

### Runs serve on dist folder

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```

### Watches for changes and recompiles in development mode

```
npm run watch
```
